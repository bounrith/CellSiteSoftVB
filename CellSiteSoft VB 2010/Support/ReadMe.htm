TRB Software® Field Photo Management (FPhotoM)

 

Welcome to the TRB Software® ReadMe file. TRB Software is a specialized software for supporting cellular industry general contractors. It is designed to streamline photo management and template reporting. To learn more about CellSiteSoft Suite and Field Photo Management (FPhotoM) please visit www.trbsoft.com/products/FPhotoM.

 

System Requirements

 

Make sure that your system meets the minimum requirements to run FPhotoM for Windows®:

 

·         1.3 gigahertz processor or higher

·         Microsoft® Windows® XP Home, Professional, or Tablet PC Edition with Service Pack 2 or 3 (32-bit and 64-bit); Windows Server® 2003 (with Service Pack 2 for 64-bit); or Windows Vista® Home Basic, Home Premium, Business, Ultimate, or Enterprise with or without Service Pack 1 (32-bit and 64-bit) , and Microsoft® Windows® 2000 Service Pack 4.

·         512 MB of installed RAM (256 MB recommended)

·         25 MB of available hard disk space (Note: The installer creates temporary files during installation so the actual hard disk space required at that time is greater than the values listed.)

·         1024 x 760 screen resolution

·         Internet Explorer 6.0

·         .NET Framework 3.5+

·         Video hardware acceleration optional
